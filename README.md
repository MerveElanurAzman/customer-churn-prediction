# Customer Churn Prediction using Machine Learning

## Project Overview

This project focuses on predicting customer churn using multiple machine learning algorithms and model optimization techniques. The main goal of the project is to analyze customer behavior patterns and identify customers who are likely to leave the service.

The project includes a complete machine learning workflow starting from data preprocessing and transformation to model training, hyperparameter optimization, and performance evaluation.

Different machine learning models were trained and compared using various evaluation metrics such as accuracy, standard deviation, ROC curves, and AUC scores.

---

## Dataset

The dataset used in this project was obtained from Kaggle and contains customer-related information used for churn prediction.

The dataset includes:
- Customer demographic information
- Service usage details
- Account and subscription information
- Churn labels

---

## Data Preprocessing

Before training the models, several preprocessing and transformation steps were applied to improve data quality and model performance.

The preprocessing stage includes:
- Handling missing values
- Encoding categorical variables
- Feature transformation
- Data normalization/scaling
- Train-test splitting

---

## Machine Learning Models

The following machine learning models were implemented and compared:

- Logistic Regression
- Bagging Classifier
- Random Forest
- XGBoost

Each model was evaluated separately to analyze performance differences and model behavior.

---

## Hyperparameter Optimization

To improve model performance, hyperparameter optimization techniques were applied using:

- GridSearchCV
- RandomizedSearchCV

The optimization methods were compared based on:
- Accuracy
- Standard deviation
- Training speed
- Overall model performance

The project also includes comparisons to determine which optimization technique performs better for each model.

---

## Performance Evaluation

The trained models were evaluated using multiple performance metrics:

- Accuracy Score
- Standard Deviation
- ROC Curve
- AUC Score

ROC and AUC analyses were performed to compare the classification capabilities of the models and measure how effectively they distinguish churned customers from non-churned customers.
